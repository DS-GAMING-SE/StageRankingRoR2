# <img src="https://github.com/DS-GAMING-SE/StageRankingRoR2/blob/master/Thunderstore/StageRankingLatest/icon.png?raw=true" width="64"> Stage Ranking

This mod adds a small results screen to the end of every stage, giving you a rank from D to S based on how well you did. This has custom UI, animations, and music, all inspired by the results screens of Sonic games. The rank you get is decided using these criteria.
| Score | Description |
| ---- | ---- |
| Time | *How quickly you beat the stage.* |
| Loot | *How many chests you've opened.* |
| Mountain Shrine | *How many mountain shrines are activated.* |

add a picture of the ui

This mod is **highly configurable**. You can change things like how many points certain critera can give you and how many points are needed for certain ranks.

The mod has **no impact on gameplay** with one tiny exception. It slightly extends the time it takes for the teleporter to take you out of each stage, but that can be disabled in the config by reducing the UI animation duration. The mod is **client-side**, meaning you can have the mod even if the people you're playing with people who don't have it.
